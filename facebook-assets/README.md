## 🚀 Quick Start Guide

### Step 1: Set Up Your Facebook Business Page (30 minutes)
1. Open `page-setup.md`
2. Follow the Facebook Page creation instructions
3. Use the provided About text and descriptions
4. Upload profile picture and cover photo (we'll create these!)
5. Set your call-to-action button to link to austinglutenfree.com

### Step 2: Install Facebook Pixel (15 minutes)
1. Open `facebook-pixel-setup.md`
2. Create your pixel in Facebook Events Manager
3. Get your pixel ID
4. **Let me know your pixel ID and I'll add it to all pages automatically!**
5. Test with Facebook Pixel Helper extension

### Step 3: Create Your First Ad Images (1-2 hours)
1. Open `ad-image-guide.md`
2. Sign up for free Canva account
3. Create 2-3 images using the templates provided
4. Start with: "40+ Restaurants" and "Pizza Guide"
5. Download as PNG files

### Step 4: Launch Your First Campaign (30 minutes)
1. Open `ad-campaign-strategy.md` for targeting details
2. Open `ad-copy.md` for ad text
3. Go to Facebook Ads Manager
4. Create "Traffic" campaign
5. Set up audience targeting (detailed in strategy doc)
6. Upload your ad images
7. Copy/paste ad copy
8. Set budget to $10/day
9. Launch! 🎉

### Step 5: Monitor & Optimize (Daily for first week)
1. Check Facebook Ads Manager daily
2. Look for CTR above 1%
3. Look for CPC below $1.50
4. After 3-4 days, pause underperforming ads
5. Increase budget on winners by 20%

## 🎯 Success Metrics to Track

### Facebook Ads Manager:
- **CTR (Click-Through Rate):** Aim for 1%+
- **CPC (Cost Per Click):** Aim for under $1.50
- **Frequency:** Keep under 3 (if higher, ad fatigue)
- **Relevance Score:** Aim for 7+ out of 10

### Google Analytics:
- **Sessions from Facebook:** Should increase week over week
- **Bounce Rate:** Aim for under 60%
- **Pages Per Session:** Aim for 2+ pages
- **Avg. Session Duration:** Aim for 1+ minutes

### Ezoic Dashboard:
- **Ad Revenue:** Should increase with more traffic
- **EPMV:** Track earnings per thousand visitors

## 💡 Pro Tips

1. **Start Small:** $10/day is enough to gather data
2. **Be Patient:** Give each ad 3-4 days before making changes
3. **Mobile First:** 80%+ of users will see ads on mobile
4. **Test Everything:** Headlines, images, audiences, times
5. **Respond Quickly:** Engage with anyone who comments on your ads
6. **Track Everything:** Use Facebook Pixel + Google Analytics together
7. **Stay Consistent:** Post on Facebook Page 3-4x per week
8. **Build Retargeting:** Your best ROI after you have site traffic
